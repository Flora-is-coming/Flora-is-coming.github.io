PHP变量

PHP 变量规则：

  • 变量以 $ 符号开始，后面跟着变量的名称
  • 变量名必须以字母或者下划线字符开始
  • 变量名只能包含字母、数字以及下划线（A-z、0-9 和 _ ）
  • 变量名不能包含空格
  • 变量名是区分大小写的（$y 和 $Y 是两个不同的变量）

PHP 没有声明变量的命令。

PHP EOF(heredoc)

PHP EOF(heredoc)是一种在命令行shell（如sh、csh、ksh、bash、PowerShell和zsh）和程序语言（像Perl、PHP、Python和Ruby）里定义一个字符串的方法

  • 以 <<<EOF 开始标记开始，以 EOF 结束标记结束，结束标记必须顶头写，不能有缩进和空格，且在结束标记末尾要有分号。
  • 开始标识不带引号与带双引号效果一致，都会解释内嵌的变量和转义符号。带单引号则不解释内嵌的变量和转义符号。
  • 当内容需要内嵌引号（单引号或双引号）时，不需要加转义符，本身对单双引号转义

PHP 类型比较

分为松散比较和严格比较

  • 松散比较：使用两个等号 == 比较，只比较值，不比较类型。
  • 严格比较：用三个等号 === 比较，除了比较值，也比较类型。

PHP 常量

特性

  1. 变性: 常量一旦定义，其值不能改变。
  2. 全局作用域: 常量在定义后，可以在整个脚本的任何地方使用(如函数体内)，无需使用 global 关键字。
  3. 数据类型: 常量的值可以是标量数据类型（如布尔值、整数、浮点数、字符串）或数组（PHP 7 及以上版本）。
  4. 区分大小写: 常量名称默认是区分大小写的。

类的定义

超级全局变量

PHP系统中自带的变量，在一个脚本的全部作用域中都可用。

$GLOBALS

$GLOBALS 是PHP的一个超级全局变量组，在一个PHP脚本的全部作用域中都可以访问。

$GLOBALS 是一个包含了全部变量的全局组合数组。变量的名字就是数组的键。

也就是把他当作数组看待，数组中有键值对，每创建一个变量，这个全局数组就会把这个变量的名字记录到数组的键里面，通过确定全局数组的键，就可以访问到这个变量的值

$_SERVER

$_SERVER 是一个包含了诸如头信息(header)、路径(path)、以及脚本位置(script locations)等等信息的数组。这个数组中的项目由 Web 服务器创建。

$_POST

使用这个全局变量，要在form（表单）元素指定method属性为post。

只能获取通过表单提交的请求体中的数据。

$_GET

使用这个全局变量，要在form（表单）元素指定method属性为get。

$_GET 收集的数据通过 URL 查询字符串传递，可以显示在浏览器地址栏

$_REQUEST

可以把它当作$_POST使用，也可以当作$_GET 使用，适用于需要访问多种来源数据，但是可能造成潜在的混淆问题，主要源于它的优先级和数据来源的重叠

  • 当POST方法和GET方法传输的参数同名时，会有优先级的问题。优先顺序是：$_POST、$_GET、$_COOKIE。这个顺序可能由于服务器设置或者配置文件发生改变。

  • 不清楚使用的是哪种方法，没有办法对这个方法可能存在的安全问题进行规避。

表单

实例

PHP 中的 $_GET 和 $_POST 变量用于检索表单中的信息，比如用户输入

表单域是指在 HTML 表单中用于输入数据的元素。有<input> <textarea> <select> <button> <datalist>等等。

每个表单域都有一个名称（name属性）。这个名称会成为数据提交时的键，用户输入的值则成为对应的值。**name 属性是唯一用于标识和提交表单域数据的必要属性**.

对表单内的元素以及属性的解析

  • <input> 标签的 name 属性用于指定表单字段的名称。当表单提交时，浏览器会使用这个名称作为键，将对应的用户输入值发送到服务器。

    简单来说就是命名传递参数的名字。

  • <input> 标签的 type 属性定义了输入字段的类型，从而决定了用户可以输入的数据类型以及输入框的外观和行为。
